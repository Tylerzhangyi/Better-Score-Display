# Better Score Display

一个用于 Yungu 学习系统的 Chrome 浏览器扩展，帮助您更好地查看和分析成绩数据。

## 项目简介

学校系统中的学情分析无法显示具体的科目分数，因此提供了这个插件来显示详细的成绩信息。插件会自动捕获 Yungu 系统中的成绩数据，并以直观的方式展示。

## 功能特性

插件根据 Yungu 官方评分标准为成绩着色：

| 等级 | 分数范围 | 颜色 | 描述 |
|------|---------|------|------|
| 5 | 95-100 | 紫色 | Extraordinary（超越） |
| 4 | 85-94 | 湖蓝色 | Advanced（精熟） |
| 3 | 70-84 | 绿色 | Mastery（掌握） |
| 2 | 60-69 | 黄色 | Developing（萌芽） |
| 1 | 50-59 | 橙色 | Passing（及格） |
| F | 0-49 | 红色 | Fail（不及格） |

## 安装步骤

### 方法一：从GitHub下载

1. **下载或克隆项目**
   ```bash
   git clone git@github.com:Tylerzhangyi/Better-Score-Display.git
   cd better-score-display
   ```
   或者直接选择 `Download ZIP`

2. **打开 Chrome 扩展管理页面**
   - 在 Chrome 浏览器地址栏输入：`chrome://extensions/`
   - 或者通过菜单：`更多工具` → `扩展程序`

3. **启用开发者模式**
   - 在扩展程序页面右上角，打开"开发者模式"开关

4. **加载扩展**
   - 点击"加载已解压的扩展程序"按钮
   - 选择项目文件夹（包含 `manifest.json` 的目录）
   - 扩展程序将自动加载

5. **确认安装**
   - 安装成功后，您会在扩展程序列表中看到 "Better Score Display"
   - 浏览器工具栏会显示扩展图标

### 方法二：Google插件商店下载

1. 在 `chrome://extensions/` 页面，点击"商店"
2. 下载项目并运行

### 使用方法

1. **访问 Yungu 系统**
   - 打开Google浏览器，访问 `https://task.yungu.org/`
   - 登录您的账号

2. **查看成绩页面**
   - 导航到综合评价,点击学业进展
   - 插件会自动在后台捕获成绩数据

3. **打开扩展弹窗**
   - 点击浏览器工具栏中的扩展图标
   - 或者使用快捷键（如果已设置）

4. **查看成绩表**
   - 弹窗中会显示当前学期的所有成绩
   - 成绩会根据 Yungu 评分标准自动着色
   - 最高分和最低分会以粗体高亮显示

### 刷新数据

如果成绩数据未显示或需要更新：

1. **强制刷新网页**
   - 在 Mac 上：按 `Command + Shift + R`
   - 在 Windows/Linux 上：按 `Ctrl + Shift + R`
   - 这会清除缓存并重新加载页面

2. **重新打开扩展**
   - 刷新页面后，再次点击扩展图标
   - 新的成绩数据会自动加载

### 数据说明

成绩表包含以下信息：
- **学期**:该成绩单的学期
- **课程名**:课程或活动的名称
- **成绩**:您的个人成绩
- **最高分**:该课程的最高分
- **最低分**:该课程的最低分
- **平均分**:该课程的平均分

## ⚠️ 注意事项

1. **首次使用**：首次安装后，需要访问 Yungu 系统的成绩页面，插件才能捕获数据
2. **数据更新**：如果成绩未显示，请使用强制刷新（Command+Shift+R / Ctrl+Shift+R）
3. **隐私安全**：所有数据仅存储在本地浏览器中，不会上传到任何服务器
4. **浏览器兼容**：目前仅支持基于 Chromium 的浏览器（Chrome、Edge、Brave 等）

## 🐛 常见问题

### Q: 扩展图标显示但点击没有反应？
A: 请确保已访问 Yungu 系统并登录账号。

### Q: 成绩数据未显示？
A: 
1. 检查是否已访问包含成绩的页面
2. 使用强制刷新（Command+Shift+R）重新加载页面
3. 重新打开扩展弹窗

### Q: 成绩颜色不正确？
A: 插件根据分数范围自动判断等级，如果分数格式异常可能导致判断错误。请检查成绩数据格式。

### Q: 如何卸载扩展？
A: 在 `chrome://extensions/` 页面找到扩展，点击"移除"按钮即可。
